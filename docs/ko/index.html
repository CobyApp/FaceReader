나의 얼굴
재해 레벨은?

아이폰 카메라 앞에 얼굴을 한 번 비추면, Apple Intelligence 가 당신을 원펀맨 세계관 풍 괴인으로 등록합니다. 늑대·호랑이·귀·용·신 5단계로 재해 등급을 판정하고, 레트로 VHS 풍의 현상수배 포스터를 발행합니다.

App Store 에서 받기 GitHub 에서 보기
▸ 주요 기능
🎯

얼굴 비율 측정

Vision 프레임워크의 랜드마크 AI 로 눈·코·입·얼굴 길이의 비율을 추출. 모든 수치는 협회 내부자료로만 처리됩니다.

🤖

AI 괴인 작명

Apple Intelligence (온디바이스) 가 세계관에 맞는 괴인 코드네임과 2줄 소개를 즉시 생성합니다.

🎴

현상수배 포스터

1080×1920px 고정 캔버스로, SNS 스토리에 그대로 공유 가능한 VHS 풍 포스터를 발행합니다.

🛟

오프라인 폴백

LLM 미지원 기기나 생성 실패 시를 대비해, 손으로 큐레이션한 괴인 도감을 언어별 100건 내장.

🌐

3개 언어 지원

한국어·일본어·영어 UI 와 포스터를 완전 다국어화. 설정에서 언제든 전환할 수 있습니다.

🔒

프라이버시 최우선

카메라 이미지와 AI 처리 모두 기기 안에서 완결. 계정 불필요, 추적 없음, 서드파티 SDK 0개.

▸ 5단계 재해 등급

협회의 위협 등급은 원펀맨의 히어로 대응 기준을 따라 5단계로 구성됩니다. 신급은 사실상 인류 종말 시나리오입니다.

늑대급C-class 대응
호랑이급B-class 대응
귀급A-class 대응
용급S-class 출동
신급인류 절망
▸ 사용법
1

얼굴 촬영

앱을 열고 화면 안내선에 따라 정면 얼굴을 한 장 촬영합니다.

2

등급 판정

Vision 이 얼굴 부위 비율을 분석해 5단계 중 하나로 분류합니다.

3

괴인 등록

Apple Intelligence 가 코드네임과 소개문을 만들어, 당신은 공식 등록 괴인이 됩니다.

4

포스터 공유

현상수배 포스터를 저장하고, SNS 에서 친구들과 재해 등급을 겨뤄보세요.

▸ 프라이버시
  • 카메라 이미지는 기기 내부에서만 처리되며, 서버로 전송하지 않습니다.
  • AI 생성 (Apple Intelligence) 도 전부 온디바이스에서 완결됩니다.
  • 계정 등록 불필요. 이메일이나 비밀번호를 요구하지 않습니다.
  • 광고·추적 SDK·서드파티 분석 도구를 일체 사용하지 않습니다.
  • 저장된 사진과 포스터는 사용자 기기 저장소에만 보관됩니다.
